Exporter - File Exporter¶

Export the trace spans to a local file.

class opencensus.trace.exporters.file_exporter.FileExporter(file_name='opencensus-traces.json', transport=<class 'opencensus.trace.exporters.transports.sync.SyncTransport'>, file_mode='w+')[source]¶

Bases: opencensus.trace.exporters.base.Exporter

Parameters:
  • file_name (str) -- The name of the output file.
  • transport (type) -- Class for creating new transport objects. It should extend from the base Transport type and implement Transport.export(). Defaults to SyncTransport. The other option is BackgroundThreadTransport.
  • file_mode (str) -- The file mode to open the output file with. Defaults to w+
